PELAJARAN 9

Pada materi dibawah ini, merupakan materi kanji yang berkaitan dengan melakukan aktivitas.
TA-TSU / たーつ

goresan : 5
arti             : berdiri
kunyomi : ta-tsu,
ta-teru
onyomi   : ritsu

kotoba :　
独立 dokuritsu / 
Independen
立場 tachiba / posisi
立候補 rikkouho / 
kandidat
創立 souritsu / 
lembaga

YASU-MU / やすーむ

goresan  : 6
arti             : istirahat
kunyomi : yasu-mu
onyomi   : kyuu

kotoba :　
休む yasumu / 
beristirahat
休日 kyuujitsu / cuti
夏休み natsuyasumi / 
libur musim panas

MI-RU / みーる

goresan : 7
arti             : melihat
kunyomi : ken
onyomi   : mi-ru/mi-eru

kotoba :
見える mieru / terlihat
見る miru / melihat
見本 mihon / sample
花見 hanami / 
melihat bunga mekar
意見 iken / pendapat
発見 hakken / penemuan

I-KU / いーく

goresan  : 6
arti             : pergi
kunyomi : i-ku,
okona-u
onyomi   : gyou, kou

kotoba :
行儀 gyougi / perilaku
行う okonau / melaksanakan
行く iku / pergi
行為 koui / tindakan

KI-KU / きーく

goresan  : 14
arti             : mendengar
kunyomi : ki-ku, ki-koeru
onyomi   : bun, mon

kotoba :　
新聞 shinbun / koran
聞こえる kikoeru / terdengar
聞く kiku / mendengar/bertanya
聞き手 kikite / pendengar

KU-RU / くーる

goresan : 7
arti             : datang
kunyomi : ku-ru
onyomi   : rai

kotoba :　
来年 rainen / tahun depan
来月 raigetsu / bulan depan
未来 mirai / masa depan
来る kuru / datang

KAE-RU / かえーる

goresan  : 10
arti             : kembali
kunyomi : kae-ru, kae-su
onyomi   : ki

kotoba :　
帰宅 kitaku / pulang kerumah
帰国 kikoku / pulang ke
negara asal
帰る kaeru / pulang
帰り道 kaerimichi / jalan kembali
