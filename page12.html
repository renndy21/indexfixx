PELAJARAN 11

Pada materi dibawah ini, merupakan materi kanji yang berkaitan dengan alam.
HAYASHI / はやし

goresan  : 8
arti             : hutan
kunyomi : hayashi
onyomi   : rin

kotoba :　
林檎 ringo / apel
林業 ringyou / 
kehutanan
森林 shinrin / hutan
松林 matsubayashi / 
hutan cemara

MORI / もり

goresan : 12
arti            : hutan
kunyomi : mori
onyomi   : shin

kotoba :
森服 morifuku / baju hutan
山森 yamamori / nama orang jepang 
森林 shinrin / hutan
森の都 mori no miyako / kota hutan

IWA / いわ

goresan : 8
arti             : batu
kunyomi : iwa
onyomi    : gan

kotoba :　
岩山 iwayama / 
gunung berbatu
岩場 iwaba / 
daerah berbatu
火山岩 kazangan / 
lava

HATAKE / はたけ

goresan : 9
arti            : ladang
kunyomi : hatake, hata
onyomi   : -

kotoba :
桑畑 kuwabatake / 
perkebunan murbai
田畑 tahata / sawah

OTO / おと

goresan   : 9
arti             : suara
kunyomi : on, in
onyomi   : oto, ne

kotoba :
発音 hatsuon / pengucapan
音楽 ongaku / musik
足音 ashioto / suara langkah kaki
音色 neiro / tone
音読み onyomi / pembaca huruf cina

AKA-RUI / あかーるい

goresan : 8
arti             : terang
kunyomi : aka-rui, aki-raka
onyomi   : mei, myou

kotoba :
証明 shoumei / verifikasi
夜明け yoake / fajar
説明 setsumei / menerangkan
明るい akarui / terang

KURA-I / くらーい

goresan : 13
arti              : gelap
kunyomi : kura-i
onyomi    : an

kotoba :
暗雲 an'un / awan gelap
暗闇 kurayami / kegelapan
暗殺 ansatsu / pembunuhan
暗号 an'gou / code,password
