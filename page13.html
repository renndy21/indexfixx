PELAJARAN 12

Pada materi dibawah ini, merupakan materi kanji yang berkaitan dengan melakukan aktivitas.
I-U / いーう

goresan : 7
arti             : berkata
kunyomi : koto, i-u
onyomi   : gen, gon

kotoba :　
言葉 kotoba / berkata
言語 gen'go / bahasa
方言 hougen / bahasa daerah
言う iu / berkata

KA-KU / かーく

goresan  : 10
arti             : menulis
kunyomi : ka-ku
onyomi   : sho

kotoba :　
絵葉書 ehagaki / 
kartu post bergambar
書く kaku / menulis
著書 chosho / buku sastra
辞書 jisho / kamus

HANA-SU / はなーす

goresan  : 13
arti           : berbicara
kunyomi : hanashi, hana-su
onyomi   : wa

kotoba :
会話 kaiwa / percakapan
電話 denwa / telepon
昔話mukashibanashi / dongeng
実話 jitsuwa / kisah nyata

TABE-RU / たべーる

ku-u
onyomi   : shoku, jiki

kotoba :

makan

makanan

nafsu makan

NO-MU / のーむ

goresan : 12
