Pada materi dibawah ini, merupakan materi kanji yang berkaitan dengan arah mata angin.
MACHI / まち

goresan : 7
arti             : kota
kunyomi : machi
onyomi   : chou

kotoba :
町長 chouchou / 
walikota
町名 choumei / nama kota
町役場 machiyakuba / 
balai kota
港町 minatomachi / 
kota pelabuhan

TERA / てら

goresan : 6
arti             : kuil
kunyomi : tera
onyomi   : ji

kotoba :　
東大寺 toudaiji / kuil toudai
山寺 yamadera / kuil gunung

KURUMA / くるま

onyomi    : sha

kotoba :
車輪 sharin / roda
自転車 jitensha / 
sepeda
電車 densha / 
kereta
車椅子 kurumaisu / 
kursi roda
