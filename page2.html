PELAJARAN 2

Pada materi ini, terdapat beberapa kanji yang berkaitan dengan hari.
NICHI / にち

goresan : 4
arti       : matahari, hari
kunyomi : hi, -ka
onyomi   : nichi, jitsu

contoh kotoba :
日曜日 nichiyoubi / minggu
今日 kyou / hari ini
日々 hibi / hari-hari
明日 ashita / besok
明後日 asatte / lusa
日記 nikki / buku harian

HI / ひ

goresan  : 4
arti             : api
kunyomi : hi, bi
onyomi   : ka

kotoba :
火曜日 kayoubi /selasa
火花 hibana / bunga api
花火 hanabi / kembang api
火山 kazan / gunung berapi
火事 kaji / kebakaran

TSUKI / つき

goresan : 4
arti             : bulan
kunyomi : tsuki
onyomi   : getsu, gatsu

kotoba :
満月 mangetsu / bulan purnama
月曜日 getsuyoubi / senin
三月 sangatsu / maret
毎月=maitsuki / setiap bulan
月明かり tsukiakari / cahaya bulan
先月 sengetsu / bulan lalu

MIZU / みず

goresan : 4
arti : air
kunyomi : mizu
onyomi : sui

kotoba :
水曜日 suiyoubi / rabu
洪水 kouzui / banjir
お水 omizu / air
水泳 suiei / berenang

KI / き

goresan  : 4
arti             : pohon
kunyomi : ki
onyomi   : moku

contoh kotoba :
木曜日 mokuyoubi / kamis
鈴木 suzuki / nama orang jepang
木の葉 konoha / daun
木材 mokuzai / kayu

TSUCHI / つち

goresan : 3
arti             : tanah
kunyomi : tsuchi
onyomi   : do, to

contoh kotoba :
土曜日 doyoubi / sabtu
土影 tsuchikage / bayangan tanah
土地 tochi / lahan
土間 doma / lantai kotor

KANE / かね

goresan  : 8
arti             : uang
kunyomi : kane, kana
onyomi   : kin, kon

contoh kotoba :
金曜日 kinyoubi / jum'at
金持ち kanemochi / kaya
お金 okane / uang
金色 kin'iro / warna emas
料金 ryoukin / ongkos

TOSHI / とし

goresan : 6
arti             : tahun
kunyomi : toshi
onyomi   : nen

contoh kotoba :　
年上 toshiue / orang yg lebih tua
去年 kyonen / tahun lalu
今年 kotoshi / tahun ini
一年 ichinen / satu tahun
